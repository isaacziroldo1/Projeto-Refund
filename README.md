# Projeto Refund

Este projeto foi desenvolvido como parte da trilha de fundamentos da
plataforma Rocketseat One, com foco na prática de HTML, CSS e JavaScript.

O objetivo é consolidar a base do desenvolvimento web por meio da criação
de uma interface funcional, aplicando conceitos essenciais de estrutura,
estilização e interatividade.

## Funcionalidades

- cadastro de novas despesas por formulário;
- formatação automática do valor para moeda brasileira (BRL);
- associação da despesa a uma categoria selecionada;
- exibição dinâmica da despesa na lista com ícone, nome, categoria e valor;
- atualização automática da quantidade total de despesas cadastradas;
- tratamento básico de erros durante a atualização da interface.

Neste estudo, foram trabalhados os seguintes pontos:
- estrutura semântica de páginas com HTML;
- organização visual e responsividade com CSS;
- manipulação do DOM e lógica de interação com JavaScript.

Este repositório representa uma etapa de aprendizado prático e evolução
contínua para projetos front-end mais completos.
